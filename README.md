# MERN boilerplate with log in

## mern_login_boilerplate
---
Fullstack MERN app featuring login, registering and session managment.

## Start DEV server
npm
```cmd
cd server
npm run start
```
yarn
```cmd
cd server
yarn dev
```

## Start DEV client server
npm
```cmd
cd client
npm start
```
yarn
```cmd
cd client
yarn start
```
## Build client
npm
```cmd
cd client
npm run build
```
yarn
```cmd
cd client
yarn build
```
